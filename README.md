# Nikel

Projeto desenvolvido como exemplo do programa [Codaí](https://growdev.com.br/codai/) da **Growdev**.

O **Nikel** é uma aplicação simples de controle financeiro pessoal.  
Com ele, é possível registrar entradas e saídas, visualizar lançamentos e acompanhar o saldo de forma prática.

---

## Tecnologias Utilizadas

- **HTML5**  
- **CSS3**  
- **JavaScript (ES6)**  
- **Bootstrap**
